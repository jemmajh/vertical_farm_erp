# Mini ERP System

## Checkpoint3 / Checkpoint6

## Current properties

The program currently implements a mini ERP system with a graphical user interface.

Implemented features:
- User login and registration
- Customer management
- Product inventory management
- Order creation with multiple order lines
- Margin calculation for order lines
- Automatic saving of data using JSON files

The system allows users to create customers, products, and quotations/orders.

Each order records the user who created it.

## Instructions

Is it possible to run the program already?  
Yes.

How is the program executed?

Run the program using:

python main.py

The graphical user interface will open and allow the user to login and use the ERP system.
